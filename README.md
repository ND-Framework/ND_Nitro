## Dependency:
[ox inventory](https://github.com/overextended/ox_inventory/releases)

#### Create this item:
```lua
["nos"] = {
    label = "NOS Bottle",
    weight = 2000,
    stack = false,
    close = true,
    client = {
        disable = { move = true, car = true, combat = true },
        usetime = 3500,
        cancel = true,
        export = "ND_Nitro.nos"
    }
},
```
